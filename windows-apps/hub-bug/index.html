Hub Bug

GitHub project management for Windows 8

Hub Bug lets you visualise GitHub issues on a Kanban style board. Arrange the board using milestones to manage your sprints and backlog, switching the board to be arranged by assigned user lets you see how your teams workload is divided up, and tben by label to see your current progress.

Kanban Board

The Hub Bug board lets you easily drag and drop issues between users, labels and milestones..

Create and Edit

Quickly create and edit issues in your GitHub repositories, assigning them to users and milestones.

Comment

Browse the details of your repositories and easily visualise how your issues are divided up over users and milestones.

Search GitHub

Need to log an issue to someone else's repository? Use the Windows 8 Search Charm to find other repositories on GitHub.
